# terminal-chat-robot
It's a terminal robot which can chat with you.

Thanks for [tuling123 API](http://www.tuling123.com/help/h_cent_webapi.jhtml?nav=doc)

## Enjoy
```bash
pip3 install terminalchatrobot
terminalchatrobot
```
